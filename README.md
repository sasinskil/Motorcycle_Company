This application helps employees with the management of a motorcycle company.

Completed:
- backend - > 100%
- frontend - > 50%

# How to start ?

## Project setup
```
npm install
```

## Compiles and hot-reloads for development
```
npm run serve
```

## Compiles and minifies for production
```
npm run build
```

# Technologies used

## Server side

- Spring Boot (Security -> JWT),
- PostreSQL


## Client side

- Vue.js,
- Bootstrap
